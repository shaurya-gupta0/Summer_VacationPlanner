Summer Vacation Planner


This is an interactive vacation planner. The user enters their name, chooses a destination, selects a hotel, enters a budget, and adds activities. The program calculates the total trip cost and creates a simple day-by-day itinerary. The user can also save the trip to a text file.

Group roles example:
Arnav: Main menu and user input
Vibushi: Destination and hotel classes
Swara: Activity and itinerary system
Shaurya: Budget calculator
Tyna: File saving and presentation/testing
